# GC Content Calculator

This project calculates the GC percentage of a DNA sequence using Python.

## What is GC Content?
GC content is the percentage of nucleotides in a DNA sequence that are either
G (Guanine) or C (Cytosine). It is commonly used in genomics and molecular biology.

## What this project does
- Takes a DNA sequence as input
- Calculates GC content
- Displays the result

- Python
- Jupyter Notebook
