## AT Content Calculator

This project calculates the AT content (percentage of Adenine and Thymine)
in a DNA sequence using Python.

## What is AT Content?
AT content is the percentage of bases in a DNA sequence that are either
A (Adenine) or T (Thymine).

## What this project does
- Takes a DNA sequence as input
- Calculates AT content percentage
- Optionally compares AT and GC content

## Tools Used
- Python
- Google Colab
